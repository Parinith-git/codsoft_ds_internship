## Task 1 · Titanic Survival Prediction

**Goal:** Build a model to predict whether a passenger on the Titanic survived or not.

- Dataset features: Age, Gender, Ticket Class, Fare, Cabin, Embarked
- Algorithms used: Logistic Regression, Random Forest
- Techniques: Missing value imputation, feature encoding, EDA

---

## Task 2 · Movie Rating Prediction with Python

**Goal:** Predict the rating of a movie based on features like genre, director, and actors.

- Dataset: Historical movie data with user/critic ratings
- Algorithms used: Linear Regression, Ridge Regression
- Techniques: Feature engineering, text encoding, correlation analysis

---

## Task 3 · Iris Flower Classification

**Goal:** Classify iris flowers into three species — *Setosa*, *Versicolor*, and *Virginica* — using sepal and petal measurements.

- Dataset: Classic Iris dataset (widely used for introductory ML)
- Algorithms used: K-Nearest Neighbors, Decision Tree, SVM
- Techniques: Data visualization, multi-class classification, model comparison

---

## Task 4 · Sales Prediction Using Python

**Goal:** Forecast the amount of a product customers will purchase based on advertising expenditure, target audience, and platform.

- Dataset: Advertising spend vs. sales data
- Algorithms used: Linear Regression
- Techniques: Correlation heatmap, feature importance, train-test split

---

## Task 5 · Credit Card Fraud Detection

**Goal:** Build a model to identify fraudulent credit card transactions.

- Dataset: Anonymized credit card transactions with class imbalance
- Algorithms used: Logistic Regression, Random Forest
- Techniques: SMOTE / undersampling, normalization, precision-recall-F1 evaluation
